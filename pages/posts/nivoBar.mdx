Nivo seems like it has several pre-fab chart components with thoughtful prop options.
One thing that seems tricky is the subtle difference between available functionality on canvas components vs svg.
Also, it seems like some charts allow for declarative "color schemes" where others (network) need functions or values.

Learning the use-cases, limitations, and affordances of multiple chart libraries can be a challenge. Subtle (or not) difference between APIs,
performance, and customizability are things to keep in mind (or not...how best to externalize this knowledge?).
